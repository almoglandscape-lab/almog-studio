# ALMOG STUDIO — אתר התיק

אתר עמוד-אחד מינימליסטי. סטטי (HTML/CSS/JS) — בלי build, בלי תלויות.

## מבנה התיקייה
```
almog-studio/
├── index.html          ← התוכן (פרויקטים, shop, קשר)
├── assets/
│   ├── style.css       ← כל העיצוב והצבעים
│   ├── script.js       ← אנימציות + החלפת צבע ניווט
│   ├── symbol.png      ← הסמל שלך (החלף בקובץ האמיתי)
│   ├── wordmark.png    ← השם שלך בכתב יד (החלף בקובץ האמיתי)
│   └── project-01.jpg  ← תמונות הפרויקטים (הוסף בעצמך)
└── README.md
```

## איך מחליפים את הלוגו (חשוב!)
כרגע יש לוגו זמני. כדי לשים את הלוגו האמיתי שלך:
1. שמור את **הסמל** בשם `symbol.png` בתיקיית `assets/` (רקע שקוף, שחור)
2. שמור את **השם** בשם `wordmark.png` בתיקיית `assets/` (רקע שקוף, שחור)
   - הקבצים פשוט יחליפו את הזמניים — אותם שמות.

## איך מוסיפים / עורכים פרויקט
ב-`index.html`, כל פרויקט הוא בלוק `<section class="project">`:
- `project-index` → המספר (01, 02, 03...)
- `project-year` → השנה האמיתית
- `project-title` → שם הפרויקט
- `project-desc` → תיאור קצר (2 שורות)
- `project-place` → מיקום · קטגוריה
- התמונה: שים קובץ ב-`assets/` ועדכן את ה-`src`

> הסדר באתר = הסדר האמיתי בחיים, ממוספר לפי תאריך.

## איך מריצים מקומית
פותחים את `index.html` בדפדפן. זהו.

## העלאה לאוויר (כשתהיה מוכן)
גוררים את כל התיקייה ל-[netlify.com/drop](https://app.netlify.com/drop) — חינם, מיידי.
דומיין מותאם אישית (almog.studio וכו') אפשר לחבר אחר כך.

## חנות (שלב הבא)
ה-section של SHOP מוכן מבחינת עיצוב. כשתחליט על פלטפורמה (Shopify / Stripe)
נחבר את המוצרים האמיתיים לתוך הרשת הקיימת.
